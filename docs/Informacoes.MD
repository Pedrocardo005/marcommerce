### Ambiente virtual

Crie um ambiente virtual:
```bash
python3 -m venv env_project
```

Para usá-lo, utilize o seguinte comando:
```bash
source env_project/bin/activate
```

Instale as dependências:
```bash
pip install -r requirements.txt
```
